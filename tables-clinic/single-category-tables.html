Single category tables

User story

"As a Welsh tax payer, I need a simple breakdown of the WRA's financial position, so I can be confident they are managing their accounts correctly."

Solution

A complex multi-category table is broken down into several simple tables by category.

Each table meets a single user need

Currency units are stated in the header cells.

Tables are given a meaningful, descriptive title.

Tables are numbered for ease of reference elsewhere in the page.
