Periodic spend tables

User story

"As a Welsh tax payer, I need to know how much money the WRA is spending on staffing costs, so I can be sure public funds are being handled sensibly."

Solution

Complex table is broken down into two simple tables, each meeting a single user need.

Double headings are condensed into a single row.

Currency units are stated in the header cells.

Each table is given a meaningful, descriptive title.

Tables are numbered for ease of reference elsewhere in the page text.

Example

People costs

Table 1 - costs by employment type for financial year 2022-23

Employment type Salaries (£) Social security (£) Other pension (£) Total paid out (£)
Permanent staff 3,627 421 1,008 5,056
Contract and agency staff 76 7 18 101
