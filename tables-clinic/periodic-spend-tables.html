Single service by area

User story

"As a resident of Wales, I need to know which child safeguarding board covers my local authority area, so that I can contact them."

Solution

User's task is reflected in a section heading starting with a verb - e.g. 'Contact...'.

The counties of Wales are presented alphabetically as sub-headings.

Under each sub-heading there is a link to the contact page for the local authority service (in this case local child safeguading board).
